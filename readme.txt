Small ColdFusion site for Relayware.
This this needs to go in a folder called "sandbox" in the webroot of your server.

The home page is:
http://yourserver/sandbox/index.cfm

Unit testing can be accessed from:
http://yourserver/sandbox/tests/TestsPageRunMe.cfm

You will need a copy of MXunit in the webroot to run the unit tests
